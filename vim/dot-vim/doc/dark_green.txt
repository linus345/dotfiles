		  Dark green color scheme    by Linus

Dark green, a color scheme with a unique touch.

==============================================================================
Color scheme options					dark_green-options*

						*g:dg_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:dg_transp_bg = 0
<
